Enrique René

Bacharel em Física. Amante da Matemática. Viciado em programação.

Sigo a vida atuando como Professor Particular e Desenvolvedor Web (Full Stack). Minha experiência com Física e Matemática me capacitam para modelagem matemática através da programação de computadores, em geral PHP, Python ou R. Modelagem computacional para cálculos estatísticos e análise de dados.

Desenvolvimento Backend: PHP 7, MySQL 8. Tenho hábito de trabalhar com as ferramentas PHPMyAdmin, Slim Framework e Laravel. Minha estratégia principal é desenvolver o sistema de API REST com rotas protegidas via OAuth, que servem conteúdo em JSON ou HTML dependendo da requisição.

Habituado a trabalhar em ambiente Linux nas distribuições Ubuntu, Debian, Fedora tanto para WorkStation quanto para Servidores. Utilizo programação Shell Script para automação de processos e assim minimizar o tempo de produção e quantidade de erros.

Desenvolvimento Frontend: HTML 5, CSS 3 e JavaScript (ES06). Trabalho com SASS, Bootstrap, React, React Native, React 360, Vue, Angular, Apache Cordova, Ionic Framework. Minha estratégia principal é o consumo de APIs através de requisições assíncronas com troca de informações via JSON.

Para desenvolvimentos rápidos e robustos buscando minimização dos custos e maior autonomia para o cliente, busco o desenvolvimento de sites em cima do WordPress (e WooCommerce), no qual trabalho com o desenvolvimento de Plugins e Temas. Desenvolvimento Mobile via WordPress é feito consumindo a WP API.
